线性代数

&||\vec u||=||\vec v||=||\vec w||\\

齐次坐标

2D point = $ (x,y,1)^T $

2D vector = $(x,y,0)^T$

所有的仿射变换(线性变换+平移)都可以写作齐次坐标的形式

且齐次坐标表示的变换矩阵最后一行一定是0,0,1，右侧n-1项各项为平移项，其余部分为之前的线性变换矩阵，三维场景中类比可得齐次变换矩阵先做线性变换再做平移

Model Transform：模型变换

View Transform：视图变换

放置相机位置以及朝向，同时定义相机的向上方向，一般放置在原点，向-z方向看，向上方向是y。

移动相机时，我们需要先平移再旋转，因为将不规则位置旋转规则化位置比较困难，考虑从规则化的位置旋转当前相机位置，得到我们所需要的旋转矩阵的逆矩阵，又因为旋转矩阵的正交性，所以我们所求的矩阵即为逆矩阵的转置

Projection：投影

即将3D中的物体投影到2D屏幕上，根据是否体现近大远小又分为：

Orthographic projection（正交投影）

放置好相机后，将z轴忽略，同时再将图象映射到$[-1,1]^3$的规范，正则（canonical）正方形里，具体的做法是将其中心移到原点处，再进行放缩

Perspeerective projection（透视投影）

将远平面上的点挤压成近平面的大小，再运用正交投影，所以透视投影的问题即转化为将远平面上的点如何挤压成近平面大小，我们规定近平面上的点永远不变，远平面在挤压时z值不变，远平面的中心点在挤压时不变

根据相似三角形，可得关系：$y’=\frac{n}{z}y$，$x’=\frac{n}{z}x$

将x’,y’代入(x’,y’,z’,1)中，齐次坐标转换变为$(x,y,z,1)^T\Rightarrow (nx/z,ny/z,unknown,1)^T$，右侧同乘z得$(nx,ny,still\ unknown,z)^T$

因此可以得到投影矩阵中的三行信息，未知量仅有四个

又因为在近平面上的点永远不变，远平面上的点z值永远不变，通过这两个条件得到等式

①由于近平面上的点永远不变，所以将z替换为n得：

因为n^2与x，y无关，从而知道了第三行前两个未知量为0

根据近平面得到的结果可以得到如下关系$An+B=n^2$

②远平面上的中心点位置不变，远平面上的z值不变，可以得到如下关系：$Af+B=f^2$

我们将MVP变换之后的$[-1,1]^3$正方体的x，y拉伸为屏幕大小，忽略z轴，再将其一角移动到原点，使整个屏幕显示在坐标轴第一象限

Sampling：采样

三角形是最基础的多边形，任何其他多边形都能拆成若干个三角形，三角形的内部一定是平面的，定义非常的清晰，只要定义三个顶点，其内部可以插值计算

将一个函数离散化的过程

判断每一个像素的中心是否在三角形内，方法即是使用沿着三角形的三条边，计算三次叉乘，来判断像素中心是否在三角形内，若在边界上则自己处理，不硬性规定

我们只需要对轴向包围盒（AABB）内的像素点进行判断

Sampling Artifacts

走样（Aliasing）现象

锯齿（Jaggies）- 空间中的采样

摩尔纹（Moire）- 图片欠采样

车轮效应（Wagon wheel effect） - 时间中的采样

本质都是因为信号变换太快，导致采样频率跟不上

同样一种采样频率，采样不同的函数，无法区分，导致走样

Antialiasing：反走样

Filtering（滤波）：将信号某一区间的频率过滤掉

傅里叶变换能将一张图片从时域变换到频域，频域的中心为低频，边缘为高频，一般图片低频信号要远多于高频信号。

我们可以在频域图中看到明显的十字，这是因为我们一般处理信号时将其视为周期性重复的信号，而图片一般情况下无重复，因为我们认为图片的左边界往外连接着右边界，即图片也是重复的，有无限多个，而左右边界一般差别较大，会有较大的信号变化。

高通滤波器，高频信息表示图像中出现较强的变化，即呈现图象中的轮廓边缘

低通滤波器，低频信息表示图像中不明显的边界特正，即图象的模糊显示

Convolution（卷积）：图形学中的卷积我们简化为运用不同的权重计算相邻信号的加权平均

Convolution Theorem（卷积定理）：时域上两个图象的卷积等于频域上其两者的乘积

Box Filter（低通滤波器），当卷积盒的大小越大时，在频域上频率越低，即过滤后的图象越模糊

Sampling = Repeating Frequency Contents

从频率的角度来说，采样就是在不断重复频域上的内容

所以走样发生的原因就是，采样的频率不够快，导致原始信号和复制的信号发生了混叠

①增加采样率

实质上增加了傅立叶域中样本之间的距离高分辨率显示器、传感器、帧缓冲器但是成本高且可能需要非常高的分辨率

②先模糊再采样

先模糊再采样，对应在频率上来说，即用低通滤波器去除原信号的高频信号，使频谱图覆盖的面积减小，从而避免混叠

一般采用1-pixel-width box filter，即每一个像素单元求其平均，从而达到模糊的效果

像素求平均的方法

ⅠMSAA(MultiSampling Anti-Aliasing)：将一个像素视作划分成许多小的块，每一个小的块来判断是否在三角形内，再将这些小的块平均起来，即得到对于该像素区域的平均近似，注意并未提高分辨率，花销是计算了更多的采样点

ⅡFXAA（Fast Approximate AA）：先得到一个有锯齿的图象，通过图像匹配找到有锯齿的边界，再更换为无锯齿的边界

ⅢTAA（Temporal AA）：复用上一帧的每个像素的值，即将MSAA的样本分布在时间上，并且当前这一帧没有引入任何新的计算量

Painter’s Algorithm 油画算法

先将最远处的物体光栅化，然后由远到近依次光栅化

问题：当在深度上存在互相遮挡，无法确定层次

Z-Buffer

为了方便理解，此处的深度我们默认为物体到摄像机的距离，即深度越小离摄像机越近

同时生成两张图：frame buffer（最终的结果图象）和depth buffer（深度图）

初始时，初始化深度图，所有的深度都是无限远的，之后将任意三角形覆盖到任意像素上，计算该像素点的深度，若该点的深度小于深度缓存中该点的深度，则将深度缓存中该点的深度更新为较小的值，该算法的时间复杂度为O（n）（n个三角形，每个三角形覆盖常数个像素），该算法的正确性与遍历三角形的顺序没有关系

Shading（着色）

着色会引入明暗和颜色的不同，对不同的物体应用不同的材质，着色具有局部性，只考虑该物体，不考虑其他任何物体的存在，所以不生成阴影

Blinn-Phong Reflectance Model（布林-冯反射模型）

Diffuse Reflection（漫反射）

Lambert’s cosine law（兰伯特余弦定理）max(0,n·l),单位面积接收到的光强为直射光强乘以夹角余弦值，而夹角余弦值为平面法向量与入射向量的点乘，当两者夹角大于90°，即从物体背面照射时，对正面不产生影响，故取0

我们假设光线能量的传播没有损失，假设初始的能量强度为I，又随着不断传播，光能所在的球体半径不断增大，而球体表面积为$4\pi r^2$，即一个距离光源距离为r的点，它到达的光强为$I/r^2$

所以漫反射最终的光照强度为：

$L_d = K_d(I/r^2)max(0,n·l)$（Kd为漫反射系数）

漫反射从任何一个角度观察得到的结果都是一致的

Specular Term（高光反射）

高光反射出现的原因是因为观测方向和镜面反射反向足够接近

在Blinn-Phong模型中，即法线与半程向量足够接近，就可观测到镜面反射，而半程向量即将入射向量与观测向量相加后归一化

则$L_s = K_s(I/r^2)max(0.cos\alpha)^p = K_s(I/r^2)max(0,n·h)^p$,其中Ks为高光反射系数

而其中的余弦值部分由于容忍度太高，导致角度较大时，仍处于一个高光的范围，所以需要加上一个指数p来限制高光的范围，一般p取100-200左右

Ambient Term（环境光照）

因为环境光的生成非常的复杂，所以我们假设场景中的任何一个点受到环境光的强度都是相同的，为$I_a$，而$K_a$为环境光系数，得到$L_a=K_aI_a$，因为每一个点的环境光都一致，所以环境光实际上为一常数，其作用是保证场景中没有点完全是黑的

最后，我们三种光照相加在一起，即为完整的Blinn-Phong反射模型

$L = L_a+L_d+L_s = K_aI_a + K_d(I/r^2)max(0,n·l) + K_s(I/r^2)max(0,n·l)^p$

Bezier curves(贝塞尔曲线)

贝塞尔曲线利用一系列控制点来定义一条曲线。

贝塞尔曲线的第一个点为曲线起始点，前两个点决定了曲线起始点位置的切线方向，最后一个点为曲线的结束点，后两个点决定了曲线结束位置的切线方向

在$b_0b_1，b_1b_2$上各取一点，分别为$b’$和$b’’$，在$b’b’’$上取一点$b’’’$

其中$b_0b’ ：b’b_1 = b_1b’’ : b’’b_2 = b’b’’’ : b’’’b’’$

让$b’$和$b’’$在相应线段上移动，取得的$b’’’$的点集即为所求曲线

$\begin{split}
&b_0^1(t)=(1-t)b_0+tb_1\\
&b_1^1(t)=(1-t)b_1+tb_2\\
&b_0^2(t)=(1-t)b_0^1+tb_1^1\\
&b_0^2(t)=(1-t)^2b_0+2t(1-t)b_1+t^2b_2
\end{split}$

四个点同理

通过定义一系列与时间有关的多项式来对不同的控制点进行插值，计算出曲线上新的点

贝塞尔曲线有以下几个性质：

对于cubic bezier(四个控制点的贝塞尔曲线)$b(0)=b_0;b(1)=b_3$

对于cubic bezier$b’(0)=3(b_1-b_0);b’(1)=3(b_3-b_2)$对贝塞尔曲线做仿射变换与对贝塞尔曲线的控制点做仿射变换后再画出对应贝塞尔曲线得到的结果一致

贝塞尔曲线不会超出给点控制点所形成的凸包的范围

Shadow Mapping

是一种图像空间的算法，并不需要知道场景的几何信息，但是会出现走样现象

核心：不在阴影中的点必须同时被摄像机和光线同时看到

首先，从光源看向场景，并记录下所能看到的物体的深度

然后我们通过摄像机来看场景，所能看到的每个点再投影回光源位置，查看光源处记录的该方向上的深度，并计算该点到光源的深度，比较两者是否相同，如果相同，则该点可见，若不相同，则该点处在阴影中

阴影可以分为软阴影和硬阴影，当一个点完全看不到光源，则为硬阴影，若只能见到一部分光源，则为软阴影，点光源一定不会出现软阴影，因为软阴影一定是因为光源有一点的大小，导致存在半影的现象

Axis-Aligned Bounding Box(AABB，轴对齐包围盒)

轴对齐包围盒即每个面都与坐标轴组成的面平行

以2D举例，将长方体的包围盒分为两个对面（slab），分别计算光线进入两个对面的点(Tmin)和离开对面的点(Tmax)，之后取Tenter=max{tmin}，Texit=min{tmax}，作为光线与包围盒的交点。

如果tenter < texit，说明光线应该是在包围盒中停留了。

同理可扩展到3D。

但是光线并不是一条直线，而是一条射线

  • 当Texit < 0 时，包围盒处于光线的后方，不存在交点
  • 当Texit >= 0 并且 Tenter < 0 ，光源本来就在包围盒中，存在交点
  • 所以，当且仅当 Tenter < Texit && Texit >= 0时，光线与包围盒相交

Monte Carlo Integration

why: we want to solve an integral, but it can be too difficult to solve analytically

how: estimate the integral of a function by averaging random samples of the function’s value

monte carlo estimator: $\int f(x)dx = \frac{1}{N} \sum_{i=1}^N \frac{f(X_i)}{p(X_i)} \quad X_i \sim p(x)$

  • the more samples, the less variance
  • sample on x, integrate on x

在积分域内不断采样，得到对应点的函数值，将其看为一常数，则将该积分化为求一矩形面积，高度为该函数值，长度为b-a，将n个采样结果相加求平均，n越大，则结果越接近实际值

Advanced Light Transport

首先要解释unbiased(无偏的)和biased(有偏的)Monte Carlo Estimators，unbiased Monte Carlo技术不会造成任何系统误差，无论使用多少个样本，无偏估计量的期望值都将始终是正确值，除此之外都属于有偏。

在有偏的情况下，有一种特殊情况，当使用接近无穷多个样本时，期望值收敛到了正确值，这种情况仍然还是有偏的，但是我们将其称为consistent(一致的)。

一种对于渲染bias更简单的理解是：

Biased == blurry(模糊)

Consistent == not blurry with infinite #samples(在无穷多个样本时不模糊)

Biased light transport methods

Photon mapping(光子映射)

这是一种有偏的且为two-stage的方法

非常适合处理Specular-Diffuse-Specular(SDS)路径并生成caustics

Stage1-photon tracing

从光源发出光子，将它们进行多次弹射，直到碰到diffuse表面时停止并记录光子

Stage2-photon collection(final gathering)

从相机子路径发射，将它们进行多次弹射，直到碰到diffuse表面时停止

分布更多光子的区域应该更亮，所以我们就来计算local density estimation(局部的密度估计)对于每一个shading point，去找到距离它最近的N个光子，并得到它们占据的面积，之后将光子数量N/面积即为local density estimation

但是为什么会是有偏的呢，实际上我们计算local density estimation，应该在当前点周围取一个微小的面积dA，来计算光子密度，而我们是通过一定光子数所占的区域来求光子密度，即：$\frac{dN}{dA} \ne \frac{\Delta N}{\Delta A}$

但从某种意义来说，如果有更多的光子被辐射出去，那么光子的密度也就更大了，寻找同样N个光子所覆盖的区域面积也就更小了，此时ΔA就更接近dA，所以，biased but consistent

我们之所以不使用固定区域寻找光子数这种方法来计算光子密度，是因为这种方法得到的结果一定是有偏的，且面积不会变小，永远不等于正确值，也就不再是consistent的方法了

是将BDPT和Photon Mapping结合起来

Key idea：如果端点不能连接但可以合并，则不要浪费BDPT中的子路径，通过使用photon mapping来处理临近“光子”的合并

有时也称为many-light方法

The Thin Lens Equation

$\frac{1}{f}=\frac{1}{z_i}+\frac{1}{z_o}$

推导：

$\frac{h_o}{z_o-f}=\frac{h_i}{f}$ $\frac{h_o}{f}=\frac{h_i}{z_i-f}$

将上两个式子做一个整理得到:

$\frac{h_o}{h_i}=\frac{z_o-f}{f}$ $\frac{h_o}{h_i}=\frac{f}{z_i-f}$

联立可得：

$\frac{z_o-f}{f}=\frac{f}{z_i-f}$

接着推得Gaussian Thin Lens Equation

$\begin{split}
(z_o-f)(z_i-f)&=f^2\\
z_oz_i-(z_o+z_i)f+f^2&=f^2\\
z_oz_i&=(z_o+z_i)f\\
\frac{1}{f}&=\frac{1}{z_i}+\frac{1}{z_o}
\end{split}$

Midpoint method/Modified Euler(中点法)

按原本的欧拉方法计算得到点a，但并不用a点，而是取起点与a点的中点b，计算中点b处的导数即速度，用b点的速度重新回到起点进行欧拉法的计算，达到c点

$\begin{split}x_{mid}&=x(t)+\frac{\Delta t}{2}\cdot v(x(t),t)\\x(t+\Delta t)&=x(t)+\Delta t\cdot v(x_{mid},t)\end{split}$

这样改良的欧拉方法将每一步的起点与终点的速度求平均，从而得到了更好的结果

$x^{t+\Delta t}=x^t+\frac{\Delta t}{2}(\dot{x}^t+\dot{x}^{t+\Delta t})$

$\dot{x}^{t+\Delta t}=\dot{x}^t+\Delta t\ddot{x}^t$

联立得：

$x^{t+\Delta t}=x^t+\Delta t\dot{x}^t+\frac{(\Delta t)^2}{2}\ddot{x}^t$

Adaptive step size(自适应改变步长)

误差估计的步长选择技术，非常实用的技术，但可能需要很小的step

首先用显示欧拉法计算Xt，步长T，再通过两次显示欧拉计算Xt/2，步长T/2

再计算误差||Xt-Xt/2||，如果该误差大于阈值，则进一步减小步长再进行同样的计算，直到误差小于阈值，算法停止

Implicit methods(隐式方法)

非正式地称为后向欧拉法，在当前步骤中，使用将来的数值

$x^{t+\Delta t}=x^t+\Delta t\dot{x}^{t+\Delta t}$

$\dot{x}^{t+\Delta t}=\dot{x}^t+\Delta t\ddot{x}^{t+\Delta t}$

解决的非线性问题，提供更好的稳定性，使用寻根算法，例如牛顿法，

那么我们很自然地想到，怎么去定义一个方法的稳定性呢？

我们使用局部截断误差(每步)/总累积误差(总体) 这样一个值来判断方法的稳定性

隐式Euler的阶数为1，这意味着：

局部截断误差为$O(h^2)O(h)$(h 是步长，即Δt)

其中O(h)的含义是，如果将h减半，那么得到的误差也将减半($O(h^2)$即降至1/4，以此类推)

Runge-Kutta Families

一系列解决ODE的高级方法，尤其擅长处理非线性问题，四阶版本是使用最广泛的版本，被称为RK4

初始条件：$\frac{dy}{dt}=f(t,y),\ \ \ \ \ y(t_0)=y_0$

RK4方法：$\begin{split}&y_{n+1}=y_n+\frac{1}{6}h(k_1+2k_2+2k_3+k_4)\\&t_{n+1}=t_n+h\end{split}$

其中：$k_1=f(t_n,y_n),k2=f(t_n+\frac{h}{2},y_n+h\frac{k_1}{2}),k_3=f(t_n+\frac{h}{2},y_n+h\frac{k_2}{2}),k4=f(t_n+h,y_n+hk_3)$

Position-based/Verlet integration

经过改进的欧拉前移后，限制粒子的位置以防止出现发散，不稳定的行为

使用受约束的位置来计算速度

这两个想法都会耗散能量，稳定性

Pros/Cons

快速且简单，但并不是基于物理的，会耗散能量(错误的)
